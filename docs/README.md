# docs/README.md

This folder is intended for lecture PDFs, DOCX files and other supporting materials.

Recommendation:
- Move large course PDFs and DOCX from the repository root into this folder to keep the root tidy.
- Example:
  git mv "Machine Learning and Data Science Fun_260218_171257.pdf" docs/
  git mv "Lesson 3 Python Libraries for data science.pdf" docs/

I did not move the files automatically to avoid creating large changes without your explicit confirmation.
